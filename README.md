Клонируем репозиторий

git clone https://github.com/traumer02/t-app-1.git
__________________________________________________________________
 Создаём .env

cp .env.example .env

Поднимаем Docker

docker compose up -d --build

docker ps

docker exec -it ..... bash
__________________________________________________________________
Запускаем миграции и сидеры

php artisan migrate

php artisan db:seed
